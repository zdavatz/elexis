Dieses File und der gesamte Inhalt des Ordners devdoc sind als Dokumentation für die Elexis Entwicklung gedacht. Dies ist nicht Teil der Dokumentation für Endanwender.

Regenenerierung

Die jeweils neueste Version dieser Dokumentation wird mit dem Ant-Script javadoc.xml aus den aktuellen Quelldateien erstellt. javadoc.xml ist im Vezeichnis doc im Elexis Projekt.

Vorgehen von Eclipse aus: Javadoc.xml mit der rechten Maustaste anklicken und Run As... Ant build auswählen.
Vorgehen von der Kommandozeile aus: ant -f javadoc.xml eingeben

In beiden Fällen wird die neueste Version im Verzeichnis devdoc erstellt.

Grundlagen

Elexis ist als Anwendung der Eclipse Rich Client Platform konzipiert. Weiterführende Dokumentation ist daher unter dem Stichtwort Eclipse RCP zu finden.

Die Ablaufumgebung ist das Java Runtime Environment Version 5 (bzw. 1.5) oder höher.

Als darunterliegende Datenbank sollte jede über Jdbc erreichbare SQL-Datenbank verwendet werden können. Auf datenbankspezifische Programmierung soll bewusst verzichtet werden. Die initiale Entwicklung wird mit den Datenbanken hsqldb, postgreSQL und mysql getestet

Es wird auch bewusst auf betriebssystemspezifische Programmierung verzichtet. Ziel ist, dass das Programm mindstens unter Windows 2000 oder höher, Mac OS X oder höher und Linux Kernel 2.6 oder höher ohne Anpassungen funktioniert.
